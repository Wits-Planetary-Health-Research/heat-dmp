# HE²AT Center Data Management Plan v4.0

This repository hosts the HE²AT Center Data Management Plan as a web page for easy provider access.

**Live site:** https://[your-username].github.io/heat-dmp/

## Quick deploy instructions

1. Create a new GitHub repository called `heat-dmp` (public)
2. Upload the contents of this folder (index.html, .nojekyll, README.md)
3. Go to **Settings > Pages** and set Source to "Deploy from a branch", branch `main`, root `/`
4. Your DMP will be live within a minute at the URL above
